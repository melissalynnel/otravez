# Otra Vez

Otra Vez is a lightweight static starter site for a vintage-inspired brand focused on wearable treasures and art consultation.

## Files

- `index.html` contains the landing page structure.
- `styles.css` contains the visual system and responsive layout.
- `main.js` adds reveal-on-scroll animation.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static file server.
